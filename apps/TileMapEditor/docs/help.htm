TileMap Editor




Instructions:

You can load and edit one of the already availabe tilemaps or you can create a new one from scratch.

Shift clicking increments the index on a tile.

Control clicking makes the current selected tile the tile you just clicked.

You can drag with the left mouse button and the tiles you draged over will become the selected ones.

Features

AStar Resize (maps are dynamic) Terrain cost (solidity)

AStar

To set a map for path finding you must edit the solidity of each tile. The solidity is a measure of how easy it is to pass through the tile. Currently the max solidity (1000) represents that a cannot be passed through at all, but a tile with a solidity of 999 can be passed through however it will this implementation of AStar will ignore it unless the alternative path has a higher cost. Use this dialog to set the solidity of the current selected tile, it will apply to all tiles with that index.
